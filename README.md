# DayZ Scripts

## Prerequisites

### Windows
```bash
pip install -r requirements.txt
```

### Linux (Ubuntu)
```bash
pip install -r requirements.txt
sudo apt install xdotool
```

## Scripts

### run.py
Auto-runner script for DayZ. Press `/` to toggle running (Shift + W), `Esc` to exit.

### Usage
```bash
python run.py
```
